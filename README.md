## Project Overview
This project is part of **Level One** of my Oasis Infobyte Data Analytics Internship.  
The goal of the project is to perform **Exploratory Data Analysis (EDA)** on a retail sales dataset to uncover patterns, trends, and insights that can help the business make informed decisions.

---

## Dataset
- **Original Dataset:** `Fashion_Retail_Sales.csv`  
- **Cleaned Dataset:** `Retail_Sales_Cleaned.csv`  
- Source: [Kaggle – Retail Sales Dataset](https://www.kaggle.com/datasets?search=retail+sales)  

---

## Project Steps
1. **Data Loading & Cleaning**  
   - Removed missing values  
   - Corrected data types  
   - Prepared dataset for analysis  

2. **Descriptive Statistics**  
   - Calculated mean, median, mode, and standard deviation  

3. **Time Series Analysis**  
   - Analyzed sales trends over time  

4. **Customer & Product Analysis**  
   - Explored customer demographics  
   - Studied purchasing behavior  

5. **Visualization**  
   - Created bar charts, line plots, and heatmaps  

6. **Recommendations**  
   - Provided actionable insights based on EDA
